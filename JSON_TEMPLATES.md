## ‚ö†Ô∏è IMPORTANT NOTES:

1. **Copy the entire JSON block** including the outer curly braces `{}`
2. **Don't copy the markdown code block markers** (```json ... ```)
3. **Make sure all strings use double quotes** `"` not single quotes `'`
4. **No trailing commas** at the end of arrays or objects
5. **All commas between properties** are required

## ‚úÖ HOW TO USE:

1. Copy one of the templates above
2. Paste it into your admin dashboard (Create Tool or Create Blog)
3. Modify the values for your specific tool/blog
4. Click "Parse & Validate"
5. Upload images
6. Click "Publish"

The system will now **automatically strip out markdown code blocks and extra text** if you accidentally include them!
